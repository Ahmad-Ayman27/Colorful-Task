# Colorful Task ✅

A lightweight and efficient task management tool designed to help users stay organized and productive.

## 🔗 Live Demo
https://subtle-travesseiro-210a13.netlify.app

## 📋 About
Colorful Task allows users to add tasks, assign due dates, and update task status with an intuitive and responsive interface. Built with a focus on simplicity and user experience.

## ✨ Features
- Add and manage tasks
- Assign due dates to tasks
- Update task status — Done, In Progress, or Pending
- Responsive design across all screen sizes
- Fast and lightweight performance

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- Tailwind CSS
- CSS
